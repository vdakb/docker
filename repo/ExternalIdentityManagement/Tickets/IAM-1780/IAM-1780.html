 Beschreibung   

Die allgemeine Spezifikation für AW-SCIMv2-Extended liegt in Confluence vor: https://confluence.bka.extrapol.de/x/eBrdFg

Zum Anheben auf v1.0 soll auch DEV bestätigen, dass für einen derartigen SCIMv2-Server eine Anbindung am F-IAM möglich sein wird. (Dass die Bedarfe der AWs abgedeckt sind, wird direkt mit denen abgestimmt.)

Basierend auf der Spezifikation sollen Beispiele abgeleitet werden, wie derart angebundene AWs über TN-SCIMv2-Extended bzgl. der Berechtigungen angesprochen werden können:

  • Abfragen der Rechtedefinitionen ohne/mit Dst-Bezug
  • Zuweisen/Entziehen der Rechte ohne/mit Dst-Bezug

Die Beispiele sollen in Confluence ergänzt bzw. die bestehenden validiert werden: 

https://confluence.bka.extrapol.de/x/jp5hG
